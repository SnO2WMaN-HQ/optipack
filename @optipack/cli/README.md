## Usage

```
$ npm install @optipack/cli --save-dev

$ yarn add @optipack/cli -D
```

```
$ optipack
```

### Option

- `-h --help`
  - display help
- `-f --file [path]`
  - _package.json_ location
- `-w`
  - overwrite _package.json_
